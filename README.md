# 🛡️ Portable Windows Compliance Auditor

> **DevSecOps Portfolio Project — Version 1.0**  
> A portable, offline Windows security compliance auditor that runs from a USB drive,
> collects system and security data from multiple machines, and generates an Excel summary report.

---

## 📋 Overview

This tool performs a **one-time security health scan** on any Windows machine without requiring:
- ❌ Internet connection
- ❌ External server or database
- ❌ Software installation on the target machine

Simply plug in the USB drive, run the auditor, and move to the next machine.
All results accumulate on the USB drive and can be exported as a single Excel report.

---

## 🏗️ Architecture

```
USB_Auditor/
├── main.py                    # Entry point & orchestrator
├── requirements.txt           # Pinned dependencies
├── config/
│   └── settings.py            # Central configuration (single source of truth)
├── scanner/
│   ├── system_info.py         # Hardware & OS metadata
│   ├── security.py            # Firewall, Defender, BitLocker, Windows Update
│   ├── services.py            # Windows Services audit
│   ├── registry.py            # Registry security checks (UAC, RDP, SMBv1)
│   └── compliance.py          # Score aggregation engine
├── exporters/
│   ├── json_exporter.py       # Per-machine JSON report
│   └── excel_exporter.py      # Aggregate all JSONs → Summary.xlsx
├── reports/                   # Auto-created: one JSON per machine
├── logs/                      # Auto-created: audit.log
└── Summary.xlsx               # Generated by --export flag
```

## ⚡ Quick Start

### 1. Prerequisites

- Python 3.12+ installed on the machine running the audit
- Run as **Administrator** for best results (some checks require elevated privileges)

### 2. Install Dependencies

```powershell
pip install -r requirements.txt
```

### 3. Run the Auditor

```powershell
# Scan this machine and save JSON report
python main.py

# Scan + export Summary.xlsx
python main.py --export

# Only export Summary.xlsx from existing reports (no scan)
python main.py --only-export
```

## 📊 What Gets Checked

### System Information
| Field | Method |
|-------|--------|
| Computer Name, Username | `socket`, `getpass` |
| Windows Edition, Build Number | `platform` |
| CPU, RAM, Disk | `psutil` |
| IP Address, MAC Address | `socket`, `uuid` |

### Security Checks (PASS / WARNING / FAIL)
| Check | Tool Used | Scoring Logic |
|-------|-----------|---------------|
| **Firewall** | `Get-NetFirewallProfile` | PASS = all 3 profiles ON |
| **Windows Defender** | `Get-MpComputerStatus` | PASS = enabled + realtime + signatures ≤ 7 days old |
| **BitLocker** | `Get-BitLockerVolume` | PASS = C: drive encrypted |
| **Windows Update** | `Get-HotFix` | PASS = last patch ≤ 30 days ago |

### Windows Services (Running / Stopped / Disabled)
| Service | Expected | Reason |
|---------|----------|--------|
| WinDefend | Running | Antivirus must be active |
| BITS | Running | Required by Windows Update |
| wuauserv | Running | Windows Update service |
| **RemoteRegistry** | **Stopped** | Remote registry access = attack vector |
| W32Time | Running | Time sync for log integrity & Kerberos |

### Registry Checks
| Key | Registry Path | Secure Value | Threat |
|-----|--------------|--------------|--------|
| **UAC** | `HKLM\...\Policies\System\EnableLUA` | `1` (ON) | Privilege escalation |
| **RDP** | `HKLM\...\Terminal Server\fDenyTSConnections` | `1` (DENY) | Brute-force / ransomware |
| **SMBv1** | `HKLM\...\LanmanServer\Parameters\SMB1Protocol` | `0` (OFF) | EternalBlue / WannaCry |

## 🧮 Compliance Score

```
PASS    = 100 points
WARNING =  50 points
FAIL    =   0 points

Score (%) = (earned points / max points) × 100

≥ 80% → Compliant    ✅
< 80% → Non-Compliant ❌
```

## 🔒 DevSecOps Design Principles Applied

| Principle | Implementation |
|-----------|---------------|
| **Fail-Safe Defaults** | UNKNOWN status = 0 points (same as FAIL) |
| **Principle of Least Privilege** | Registry opened with `KEY_READ` only |
| **Immutable Audit Trail** | JSON files are never overwritten (timestamp in filename) |
| **Non-repudiation** | UTC timestamp + username recorded in every report |
| **Defense in Depth** | Multiple check layers: Security + Services + Registry |
| **Separation of Concerns** | Each scanner module owns exactly one domain |

## 🔧 Exit Codes

| Code | Meaning |
|------|---------|
| `0` | Scan completed — machine is **Compliant** |
| `1` | Scan completed — machine is **Non-Compliant** |
| `2` | Fatal error — scan could not complete |
